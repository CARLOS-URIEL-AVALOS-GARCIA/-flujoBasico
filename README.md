# 1. ¿Qué es una Base de Datos Relacional?
Es un tipo de base de datos que organiza la información en tablas relacionadas entre sí. Usa relaciones lógicas entre los datos para evitar equivocaciones y mantener la integridad de la información.

# 2. ¿Qué es una Tabla en el contexto de Bases de Datos Relacionales?
Es una estructura que almacena datos de un mismo tipo de entidad por ejemplo, clientes o productos. Está formada por columnas y renglones para organizarse mas facilmente

# 3. ¿Qué es una relación en el contexto de Bases de Datos Relacionales?
Es el vínculo lógico entre dos o más tablas, normalmente estan organizadas a través de claves primarias y claves foráneas, que permite asociar datos que esten relacionados.

# 4. ¿Qué representa una columna en el contexto de Bases de Datos Relacionales?
Representa un atributo o campo de la entidad.Tambien define cual es el tipo de datos que se almacenarán algunos ejemplos como: nombre, edad, precio, etc.

# 5. ¿Qué representa un renglón en el contexto de Bases de Datos Relacionales?
Representa un registro o instancia única de la entidad. Es decir, un conjunto completo de datos para un solo elemento específico por ejemplo, un cliente.

# 6. ¿Qué es un motor de base de datos relacional?
Es el software encargado de crear, administrar y consultar bases de datos relacionales, manejando almacenamiento, seguridad, transacciones y ejecución de consultas SQL.

# 7. ¿Cuáles son los motores de bases de datos relacionales más populares?
Algunos de los más usados son:
-MySQL
-PostgreSQL
-Oracle Database
-Microsoft SQL Server
-MariaDB

# 8. ¿Cuál es el motor de base de datos ligero que no necesita un servidor?
El motor de base de datos ligero que no necesita un servidor es SQLite.
SQLite es un sistema de gestión de bases de datos relacional embebido, lo que significa que no funciona como un servicio independiente ni requiere un servidor en ejecución.

# 9. ¿Qué significa SQL en el contexto de bases de datos relacionales?
SQL significa Structured Query Language Lenguaje de Consulta Estructurada.
Es el lenguaje estándar para crear, consultar, modificar y administrar bases de datos relacionales.

# 10. ¿Cuáles son los elementos de un sistema de información basado en bases de datos relacionales?
-Incluye principalmente:
-Base de datos (tablas y relaciones)
-Motor de base de datos
-Lenguaje SQL
-Aplicación o sistema que usa la base de datos
-Usuarios
-Hardware y sistema operativo
